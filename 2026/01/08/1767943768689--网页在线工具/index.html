✅ 网页在线工具

发表于 2026-01-09 标签： 工具

在线工具汇总

1️⃣ Markdown 转 HTML 工具（及其他 Markdown 工具）

  • Markdown 转 HTML
    • https://www.markdownlang.com/ja/markdown-to-html/

📌 用途说明：
将 Markdown 文本快速转换为 HTML，适合文档预览、博客发布、设计书输出等场景。

2️⃣ FlowChart 流程图工具（Mermaid）

  • Mermaid 在线编辑器（FlowChart）
    • https://mermaid.live/edit

📌 用途说明：
使用 Mermaid 语法在线编写流程图、时序图、架构图，支持实时预览与编辑，常用于技术文档和设计说明。

3️⃣ BPMN 在线工具

  • BPMN.io 在线编辑器
    • https://demo.bpmn.io/

📌 用途说明：
在线创建、查看和编辑符合 BPMN 2.0 标准的业务流程图，可直接用于 Camunda / Flowable / Activiti 等流程引擎。

4️⃣ Mermaid → 图片 在线转换工具

  • Mermaid2Img

    • https://mermaid2img.com/

  • Mermaid Converter（日本站）

    • https://www.mermaid-converter.jp/

  • Simple Mermaid Tool

    • https://www.simplemermaid.com/mermaid-tool.html

📌 用途说明：
将 Mermaid 图表代码转换为 PNG / SVG 等图片格式，适合插入 PPT、Word、设计书或对外资料。

📎 总结对比

分类 主要用途 适合场景
Markdown → HTML 文本转网页 文档发布、说明书
Mermaid FlowChart 文本生成流程图 技术设计、流程说明
BPMN 工具 业务流程建模 审批流、系统流程
Mermaid → 图片 图表导出 PPT、设计书

🏆 最推荐 TOP 20 在线工具（按重要度排序）

评选依据：日本现场使用率 + 实用性 + 学习成本 + 通用性
目标：真正能在项目和现场长期使用的工具

🥇 TOP 1–5（核心必备 · 强烈推荐）

1️⃣ draw.io（diagrams.net）

  • 🔗 https://app.diagrams.net/
  • 用途：流程图 / 架构图 / 系统图
  • 特点：日本现场使用率极高，几乎是“默认标准工具”

2️⃣ Markdown 编辑 / 转换工具

  • 🔗 https://www.markdownlang.com/ja/markdown-to-html/
  • 用途：Markdown → HTML
  • 特点：设计书、说明文档高频使用

3️⃣ Mermaid Live

  • 🔗 https://mermaid.live/edit
  • 用途：文本生成流程图 / 时序图
  • 特点：代码即文档，程序员友好

4️⃣ BPMN.io

  • 🔗 https://demo.bpmn.io/
  • 用途：BPMN 业务流程图建模
  • 特点：Camunda / Flowable / Activiti 必备

5️⃣ Notion

  • 🔗 https://www.notion.so/
  • 用途：文档 / 知识库 / 协作
  • 特点：个人与团队生产力核心
