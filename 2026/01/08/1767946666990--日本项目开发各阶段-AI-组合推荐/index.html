🇯🇵 日本项目开发各阶段 × AI 组合推荐

适用于 日本 IT 现场（SIer / 受託开发 / 社内系统）
覆盖：要件定义 → 基本设计 → 详细设计 → 实装 → 测试 → 发布 → 运维
重点：设计书、沟通成本、质量、效率

① 要件定义（要件定義）

🎯 主要课题

  • 客户日文表达模糊
  • 需求反复、遗漏
  • 要件书难以结构化

🤖 AI 组合推荐

  • ChatGPT / Claude
  • Perplexity
  • Notion AI

🧩 使用方式

  • ChatGPT
    → 日文要件整理、模糊表达澄清、Q&A 模板生成
  • Claude
    → 长要件文档结构化（章立て、表格）
  • Notion AI
    → 要件定义书（要件一覧 / 非功能要件）

📄 成果物示例：

  • 要件定義書
  • 业务流程概要说明

② 基本设计（基本設計）

🎯 主要课题

  • 业务流程复杂
  • 设计书说明性不足
  • 审查（レビュー）成本高

🤖 AI 组合推荐

  • ChatGPT
  • Mermaid Live
  • BPMN.io
  • Notion AI

🧩 使用方式

  • ChatGPT
    → 从要件生成基本设计说明（日文）
  • Mermaid
    → 画系统结构图、处理流程图
  • BPMN.io
    → 业务流程（AS-IS / TO-BE）
  • Notion AI
    → 基本设计书整合

📄 成果物示例：

  • 基本設計書
  • BPMN 业务流程图
  • 系统构成图

③ 详细设计（詳細設計）

🎯 主要课题

  • 设计遗漏
  • IF 定义不清
  • Java 设计与设计书脱节

🤖 AI 组合推荐

  • ChatGPT
  • Claude
  • Mermaid
  • draw.io

🧩 使用方式

  • ChatGPT
    → 方法设计、异常设计、表结构说明
  • Claude
    → 大量详细设计书校对
  • Mermaid
    → 时序图（Sequence Diagram）
  • draw.io
    → 类图、ER 图

📄 成果物示例：

  • 詳細設計書
  • IF 定義書
  • 时序图 / 类图

④ 实装（製造 / 実装）

🎯 主要课题

  • Java 代码量大
  • 现场规范复杂
  • 学习成本高

🤖 AI 组合推荐

  • GitHub Copilot
  • ChatGPT
  • Perplexity

🧩 使用方式

  • Copilot
    → Java / Spring / SQL 自动补全
  • ChatGPT
    → 代码解释、重构、单元逻辑说明
  • Perplexity
    → 查框架用法、版本差异

📄 成果物示例：

  • Java 源码
  • Mapper / SQL
  • 单元说明注释

⑤ 单体测试（単体テスト）

🎯 主要课题

  • UT 编写耗时
  • 覆盖不足
  • Case 漏写

🤖 AI 组合推荐

  • ChatGPT
  • Copilot
  • Excel + Copilot

🧩 使用方式

  • ChatGPT
    → 自动生成 UT Case（正常 / 异常）
  • Copilot
    → JUnit / Mockito 编写
  • Excel Copilot
    → 测试用例整理

📄 成果物示例：

  • 单体测试规格书
  • JUnit 测试代码

⑥ 结合 / 系统测试（結合・総合テスト）

🎯 主要课题

  • 测试观点评估难
  • Case 规模大
  • 文档维护成本高

🤖 AI 组合推荐

  • ChatGPT
  • Notion AI
  • Excel Copilot

🧩 使用方式

  • ChatGPT
    → 测试观点评估表
  • Notion AI
    → 测试说明书整理
  • Excel Copilot
    → 结果统计、缺陷分析

📄 成果物示例：

  • 结合测试仕様書
  • 测试结果报告

⑦ 发布（リリース）

🎯 主要课题

  • 手顺遗漏风险
  • 文档不统一

🤖 AI 组合推荐

  • ChatGPT
  • Notion AI

🧩 使用方式

  • ChatGPT
    → 发布手顺自动生成
  • Notion AI
    → 发布检查清单

📄 成果物示例：

  • リリース手順書
  • チェックリスト

⑧ 运维 / 保守（運用・保守）

🎯 主要课题

  • 障害调查慢
  • 文档缺失
  • 知识传承困难

🤖 AI 组合推荐

  • ChatGPT
  • Notion AI
  • Perplexity

🧩 使用方式

  • ChatGPT
    → 日志分析、障害原因整理
  • Notion AI
    → 运维手册、FAQ
  • Perplexity
    → 查外部案例

📄 成果物示例：

  • 障害報告書
  • 运维手册
