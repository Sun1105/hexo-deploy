📘 ～一方だ / ～ばかりだ / ～続ける / ～つつある

发表于 2025-12-11 标签： 日语语法

四个表达的比较

1. ～一方だ

  • 接续：动词辞书形 + 一方だ
  • 含义：情况持续单向发展，通常不可逆。
  • 语感：偏书面，常用于负面趋势。
  • 例句：物価は上がる一方だ。 （物价不断上涨。）

2. ～ばかりだ

  • 接续：动词辞书形 + ばかりだ
  • 含义：情况持续发展，常用于负面。
  • 语感：比「一方だ」更强烈，带有抱怨或无奈。
  • 例句：事故が増えるばかりだ。 （事故越来越多。）

3. ～続ける

  • 接续：动词连用形 + 続ける
  • 含义：动作或状态持续进行。
  • 语感：中性，强调动作的持续性。
  • 例句：勉強を続ける。 （继续学习。）

4. ～つつある

  • 接续：动词连用形 + つつある
  • 含义：某种变化正在进行中。
  • 语感：书面语，强调过程。
  • 例句：技術は進歩しつつある。 （技术正在进步。）

📊 对比表

表达 接续形式 含义 语感 例句
～一方だ 动词辞书形 趋势单向发展 偏书面，常负面 物価は上がる一方だ
～ばかりだ 动词辞书形 越来越…（多负面） 强烈负面，带抱怨 事故が増えるばかりだ
～続ける 动词连用形 持续进行 中性，动作延续 勉強を続ける
～つつある 动词连用形 正在变化中 书面语，强调过程 技術は進歩しつつある

✅ 总结

  • 一方だ → 趋势不可逆，常负面。
  • ばかりだ → 越来越…，更强烈负面。
  • 続ける → 动作持续进行，中性。
  • つつある → 正在变化中，书面语。
